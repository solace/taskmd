# Worklog: 174 - Set up e2e test foundation and helpers

## 2026-02-20T10:00:00Z

Starting implementation of the e2e test foundation.

**Approach:** Create a new `internal/e2e/` package with:
- `TestMain` that builds the binary once into a temp dir
- `run`/`mustRun` helpers for subprocess invocation
- `writeTask` helper for creating test task files
- Environment isolation via HOME override and NO_COLOR=1
- Makefile targets: `make e2e` (runs e2e only), `make test` unchanged (excludes e2e)
- Smoke test: `taskmd --help` exits 0 with usage text

**Key decisions:**
- Using build tags (`//go:build e2e`) to separate e2e from unit tests — this means `go test ./...` naturally excludes e2e without needing path exclusions
- `make e2e` uses `go test -tags e2e -count=1 ./internal/e2e/...`
